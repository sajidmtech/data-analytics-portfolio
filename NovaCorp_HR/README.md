# NovaCorp HR Analytics

## Overview
A beginner HR analytics project built from scratch to learn 
the end-to-end data analyst workflow using a fictional 
company called NovaCorp.

## Tools Used
- MySQL / DataGrip — data querying
- Excel — data cleaning and pivot tables
- Power BI — dashboard and visualization

## Business Questions
1. Who works here? — employee listing by department
2. What are we paying per team? — salary analysis by department
3. Who has been here the longest? — tenure calculation
4. Where is attrition happening? — departure analysis

## Key Skills Demonstrated
- SELECT, WHERE, ORDER BY
- GROUP BY, AVG, COUNT
- JOIN and LEFT JOIN
- CASE WHEN
- Subqueries
- VLOOKUP and AVERAGEIF in Excel
- Pivot tables
- Power BI bar charts and dashboards
